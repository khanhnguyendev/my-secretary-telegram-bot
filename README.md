# Calendar Telegram Bot

Create Google Calendar events via Telegram.

## Required ENV

BOT_TOKEN=telegram_bot_token  
CALENDAR_ID=google_calendar_id  
SERVICE_ACCOUNT_JSON=base64_json  
ALLOWED_USERS=telegram_user_id_1,telegram_user_id_2

## Input Format

16-18h: cầu lông  
mai 9-10: họp team @Online
